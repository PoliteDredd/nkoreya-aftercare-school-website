# Nkoreya Tutoring Service Website

A modern full-stack tutoring platform built to connect students with tutoring services through a responsive, user-friendly, and scalable web application.

The project focuses on delivering a smooth digital experience for students and tutors while implementing modern software development practices, secure authentication, real-time backend services, and responsive UI/UX design.

## 🚀 Tech Stack

### Frontend

* React 18
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui
* React Router DOM
* TanStack Query
* Zod & React Hook Form
* Lucide React

### Backend

* Supabase
* PostgreSQL
* Authentication
* Realtime Subscriptions
* Storage Services
* Row Level Security (RLS)

## 🌐 Deployment

* Vercel (Frontend Hosting)
* Supabase Cloud (Backend Hosting)
* Domain managed via Domains.co.za
